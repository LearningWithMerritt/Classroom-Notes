# `Defining Copyright`
`Copyright `is a legal right that automatically protects original works of authorship, including software. 
* The creator has exclusive rights to copy, distribute, modify, and publicly display their work. Others cannot do so without permission. 

More at [copyright.gov](https://www.copyright.gov/what-is-copyright/)

# `Legal Use`
`Legal use` of copyrighted work is possible, through permissions, licensing terms, and the concept of fair use.

<br>

`Licensing` is a legal agreement where the copyright owner gives permission to someone else to use their work under specific conditions.

<br>

`Permissions` are direct approvals from the copyright owner that allow you to do something that would otherwise be illegal (like copying, modifying, or distributing).

<br>

`Fair Use` is a legal doctrine that allows limited use of copyrighted material without permission — for specific purposes like:
* Criticism
* Commentary
* News reporting
* Teaching
* Scholarship
* Research

# `Licenses`
`Licensing` is a legal agreement where the copyright owner gives permission to someone else to use their work under specific conditions.

<br>

## `Creative Commons`
Creative Commons licenses grant public permission to the the creative work under copyright law.
* These licenses answer the question: What can I do with this work?
* Usually have terms and conditions.

More at : [creativecommons.org](https://creativecommons.org/share-your-work/cclicenses/)

There are six different Creative Commons Licenses:
| License | Name | Brief Description |
|:--------|:-----|:-------------------|
| `CC BY` | Attribution | You can copy, use, and modify the work — even commercially — as long as you give credit. |e
| `CC BY-SA` | Attribution-ShareAlike | You can modify and use the work — even commercially — but you must credit and license your new work under the same terms. |
| `CC BY-ND` | Attribution-NoDerivs | You can use the work — even commercially — but no changes can be made, and credit must be given. |
| `CC BY-NC` | Attribution-NonCommercial | You can modify and use the work non-commercially, with credit; commercial use is not allowed. |
| `CC BY-NC-SA` | Attribution-NonCommercial-ShareAlike | You can modify and use the work non-commercially, must credit, and license new work under identical terms. |
| `CC BY-NC-ND` | Attribution-NonCommercial-NoDerivs | You can share the work non-commercially with credit, but no modifications or commercial use allowed. |

## `Open Source Software Licenses`

| License | Brief Description |
|:--------|:-------------------|
| `MIT License` | Very permissive; allows almost anything with attribution. |
| `GNU GPL (General Public License)` | Requires derivative works to also be open-source ("copyleft"). |
| `Apache License 2.0` | Allows use with patent rights and modifications, requires attribution. |
| `BSD License` | Very simple and permissive; minimal restrictions. |
| `Mozilla Public License (MPL)` | Allows modifications but must share changes under the same license. |

## `Public Domain and Dedication Licenses`

| License | Brief Description |
|:--------|:-------------------|
| `CC0 (Creative Commons Zero) `| Waives all rights; work is free to use without permission. |
| `Public Domain Mark` | Labels works that are already in the public domain (no copyright). |

## `Proprietary / Commercial Licenses`

| License | Brief Description |
|:--------|:-------------------|
| `EULA (End User License Agreement)` | Standard software agreement; limits use, non-transferable. |
| `SaaS (Software as a Service) Licenses` | Rent software access on a subscription basis. |
| `Single-User / Multi-User Licenses` | License based on number of allowed users. |

## `Media and Content Licenses`

| License | Brief Description |
|:--------|:-------------------|
| `Royalty-Free License` | Pay once, use multiple times without extra fees. |
| `Rights-Managed License` | Pay based on specific use (e.g., duration, region, platform). |
| `Copyleft License` | Must share derivative works under the same license. |

## `Academic and Research Licenses`

| License | Brief Description |
|:--------|:-------------------|
| `Open Access Licenses (e.g., CC-BY for papers)` | Academic works freely available for reading, sharing, and sometimes reuse. |
| `Data Commons Licenses` | Allows free sharing and reuse of scientific datasets. |

# `Open Source`
`Open source` refers to software where the source code is freely available for anyone to use, modify, and distribute. 

<br>

Common Open Source Licenses:w
* MIT License
* GNU GPL
* Apache License


## `Freeware and Shareware`
Freeware and Shareware are free to use but not open source.
* Freeware: 100% free to use but no access to source code (e.g., Zoom, Skype).
* Shareware: Free for trial, then you have to pay to continue (e.g., WinRAR).
